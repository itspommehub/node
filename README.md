## ✨ 项目简介

这里不是一份冷冰冰的技术文档，  
而是一次对 **网络世界底层路线的探索** 🌐。  
从 BGP 到 IEPL，从 Shadowsocks 到 Trojan ——  
每一条线路、每一个协议，  
都在试图回答一个问题：  
> 🔍 *「怎样才能让世界更近一点？」*  

---

## 📚 内容一览

- 🌊 节点类型全解析（BGP / IEPL / IPLC）  
- 🔒 协议区别对比（Trojan / Shadowsocks / Vmess）  
- ⚙️ 延迟、稳定性、隐私性多维分析  
- 💡 实用选择建议与使用技巧

## 🧭 一、线路类型速查表

“线路” 决定了节点的速度、稳定性与抗封锁能力。

|   |   |   |   |   |
|---|---|---|---|---|
|🏷️ 类型|🌐 全称|💬 说明|⚖️ 优缺点|🎯 适用场景|
|🌀 **中转线路**|Relay / Proxy Transit|数据先经过国内中转服务器再出国|✅ 成本低  <br>⚠️ 稳定性一般|普通上网、轻度使用|
|🌏 **BGP 动态线路**|Border Gateway Protocol|自动选择最优出口路由|✅ 延迟低  <br>❌ 成本略高|通用日常使用|
|⚡ **IPLC 专线**|International Private Leased Circuit|点对点跨境物理专线|✅ 极稳无丢包  <br>❌ 昂贵|企业远程、跨境办公|
|🛠️ **IEPL 专线**|International Ethernet Private Line|企业级以太网专线|✅ 延迟极低、超稳  <br>❌ 成本高|游戏、视频会议|
|🧩 **BGP + IEPL 混合**|Hybrid Line|动态路由 + 专线结合|✅ 灵活稳定  <br>❌ 依机场质量|高端用户|
|🌐 **Anycast 全球路由**|Global Routing|全球共享IP，自动就近连接|✅ 可用性高  <br>❌ 波动大|多地区访问|
|🧱 **原生线路**|Native IP|真实当地IP地址|✅ 解锁流媒体、ChatGPT  <br>❌ 成本高|影音、AI用户|
|☁️ **云节点**|AWS / GCP / Azure|部署在云平台|✅ 成本低  <br>❌ 稳定性中等|测试、轻度使用|
|📡 **CN2 / CMI 优化**|运营商专线|电信/移动优化路径|✅ 延迟低  <br>❌ 区域依赖强|电信/移动用户优先|

## ⚙️ 二、节点标签释义

|   |   |   |
|---|---|---|
|🔖 标签|💡 意思|📘 说明|
|🧩 **原生**|地区真实IP|可解锁Netflix、Disney+、ChatGPT等|
|🔁 **中转**|国内服务器中继出国|成本低、易波动|
|🧠 **BGP动态**|自动最优出口|稳定、路由智能|
|🛰 **IEPL/IPLC**|企业级专线|几乎无丢包、极稳|
|🌍 **Anycast**|全球共享IP|自动匹配最近节点|
|☁️ **AWS / GCP / Azure**|云平台节点|通用但延迟略高|
|⚡ **直连**|无中转直连国外|延迟低但易封锁|

## 🔐 三、代理协议对比

不同协议代表不同的加密与伪装方式。

|   |   |   |   |   |   |
|---|---|---|---|---|---|
|🚀 协议|🔤 全称|✨ 特点|👍 优点|⚠️ 缺点|💡 适用人群|
|💨 **Shadowsocks (SS)**|轻量加密代理|速度快、兼容广|✅ 延迟低|❌ 抗封锁弱|普通用户|
|🔰 **ShadowsocksR (SSR)**|SS 改进版|混淆防检测|✅ 抗封锁略强|❌ 老旧|老机场|
|🧩 **Vmess (V2Ray)**|V2Ray核心协议|可自定义伪装|✅ 稳定强大|❌ 稍复杂|技术用户|
|🚀 **Vless (V2Ray)**|无状态轻量协议|性能高、低延迟|✅ 主流稳定|❌ 需TLS支持|主流机场|
|🛡 **Trojan**|TLS伪装协议|模拟HTTPS流量|✅ 抗封锁最强|❌ 稍重|大陆用户首选|
|⚡ **Hysteria / Hysteria2**|UDP加速协议|面向高延迟网络|✅ 极速、低延迟|❌ 客户端少|游戏/远程桌面|
|🔮 **Tuic**|新一代UDP协议|高效加密、延迟低|✅ 极速|❌ 兼容性差|新兴高端协议用户|

## 📶 四、选节点实用建议

|   |   |   |
|---|---|---|
|🎯 使用场景|🚀 推荐线路|🔐 推荐协议|

|💻 企业远程办公|IPLC 专线|Trojan / Hysteria|

## 🧠 五、术语小百科

|   |   |
|---|---|
|📘 术语|🔍 含义|
|🧩 **原生IP**：真实地区分配IP，可解锁本地内容||
|🌀 **中转节点**：流量经国内跳板转发出国||
|🌐 **直连节点**：直接出国线路，易被墙||
|⚙️ **订阅链接**：机场配置自动更新地址||
|🔁 **测速模式**：Ping / TCP / HTTP 不同测试方式||
|🧱 **被墙**：节点封锁或识别失败||
|🧭 **延迟**：单位 ms，<150ms 即体验流畅||

## 🔧 七、实用贴士

- 🔄 **定期刷新订阅**，保证最新节点信息；
- ⚙️ **测速建议用 HTTP 模式**，更准确；
- 🌐 **Trojan + IEPL** 是目前最稳组合；
- ☁️ 机场中转线路便宜但可能波动；
- 🧱 若节点全超时，多半是订阅过期或服务器维护；
- 📱 蜂窝网络测速更真实，Wi-Fi 有时会屏蔽 ICMP。

## ✨ 八、小结

🧭 如果你是：

- 普通用户 → BGP + Shadowsocks
- 稳定党 / AI 用户 → IEPL + Trojan
- 游戏玩家 → IEPL / Hysteria2
- 想解锁流媒体 → 原生IP节点

🪶 **一句话总结：**

💡 “BGP 是聪明的，IEPL 是稳重的，Trojan 是隐秘的。”  
选节点，就像选车——**要看用途，不只看速度。**
